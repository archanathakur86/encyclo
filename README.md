# Encyclopedia of Animals

Encyclopedia of Animals is an interactive and educational website built using HTML and CSS. The project is designed to provide an engaging way to explore the animal kingdom while keeping the interface simple, clean, and easy to navigate.

The website allows users to explore different categories of animals such as reptiles, birds, and mammals, and learn about their characteristics, habits, and unique features. The main focus of this project is to present information in a visually appealing and user-friendly manner.

## Overview

This project was created to practice frontend fundamentals such as semantic HTML structure, responsive layouts, and clean CSS styling. It also aims to promote awareness about biodiversity by offering an informative platform for students, educators, and nature enthusiasts.

## Key Features

- User-friendly and clean interface for smooth navigation  
- Categorized animal sections including reptiles, birds, and mammals  
- Informative content displayed in a structured and readable format  
- Responsive design suitable for desktop, tablet, and mobile devices  
- Visually engaging layout focused on readability and accessibility  

## Tech Stack

- HTML5 – For structuring the content  
- CSS3 – For styling, layout, and responsiveness  

## Project Focus

The primary focus of this project was to strengthen frontend fundamentals, improve layout design skills, and build a responsive informational website without using external frameworks or libraries.
